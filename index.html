What's new ?

Lightning Shaders

An interactive environment for learning GLSL via WebGL. Users can recompile, edit, and save their shader programs. Interface thoughtfully built with React and served with Express, Node, and MongoDB backend.

Gengo At Vista

Gengo At Vista is a language learning interface idea inpired by the book Spanish At Sight. It is built with React, hosted on Heroku and served on top of a Node backend.

Mernboiler

A MERN stack web application boilerplate written in Bash. One script creates an application architecture that utilizes Node, Express, MongoDB, and React.

Me

I'm a software developer and visual artist based in New York. With versatility as a natural strength, code and creative design are pursuits that coalesce well for me. They are skills that I strive to continuously hone. Projects that I enjoy working on include building user interfaces, streamlining the display of data, editing digital content, and, recently, configuring backend architecture. Although I feel most comfortable on the frontend, I'm drawn to working across the entire stack and feel that I excel in a Full Stack position. When I'm not coding I'm usually socializing, making music, or adventuring outside. Oftentimes I explore new places to grow my photography portfolio. Traveling is a big part of my life, too, and I'm always looking to expand my network. Let's connect and collaborate!
